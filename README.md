# П412NEW - Класс String для C++

Проект реализует собственный класс `String` для работы со строками в C++.

## Описание

Класс `String` предоставляет функциональность для работы со строками, аналогичную стандартному `std::string`, но реализованную с нуля.

## Основные возможности

- Конструкторы: по умолчанию, из C-строки, из символа, копирующий
- Операторы присваивания
- Операторы доступа по индексу `[]`
- Методы работы со строками:
  - `size()` - размер строки
  - `empty()` - проверка на пустоту
  - `c_str()` - получение C-строки
  - `clear()` - очистка строки
  - `push_back()` / `pop_back()` - добавление/удаление символов
  - `substr()` - подстрока
  - `find()` - поиск символа или подстроки
  - `starts_with()` / `ends_with()` - проверка начала/конца строки
  - `to_upper()` / `to_lower()` - преобразование регистра
  - `trim()` - удаление пробелов
  - `reverse()` - обращение строки
- Операторы конкатенации `+` и `+=`
- Операторы сравнения `==`, `!=`, `<`, `<=`, `>`, `>=`
- Операторы ввода/вывода `<<` и `>>`

## Структура проекта

- `String.h` - заголовочный файл с объявлением класса
- `String.cpp` - реализация методов класса
- `П412NEW.cpp` - файл с тестами и примером использования
